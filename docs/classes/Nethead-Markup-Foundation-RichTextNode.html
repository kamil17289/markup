RichTextNode.php : 13

Creates rich formatted texts in HTML.

It has many chainable methods to make it easier and quicker to create many tags inside and format the text as you need, using bold, underline, italics, superscripts and other formatting options. It is like a factory, but adds all its children to internal storage, so they can be quickly rendered at once when you cast it to string.
