TextNode.php : 15

Class TextNode is used for adding texts inside of HTML tags.

You can use simple strings as a child elements, but everywhere you need the input to be sanitized before outputting, use this. TextNode always escapes it's contents, no matter if render() method or (string) cast is used. It also contains a few useful helper methods.
